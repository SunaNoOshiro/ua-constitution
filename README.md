# ua-constitution

A modern web app for reading, searching, and navigating the Constitution of Ukraine in a clean and user-friendly interface.

## Why

The Constitution of Ukraine is usually available as plain text, legal pages, or PDF documents. These formats are not always comfortable for quick reading, mobile use, search, or structured navigation.

This project aims to provide a better reading experience for citizens, students, researchers, journalists, and anyone who wants quick access to the constitutional text.

## Features

* Clean article-based reading experience
* Fast search by keywords and article numbers
* Easy navigation by chapters and sections
* Mobile-friendly layout
* Bookmarks for important articles
* Clear typography for long-form legal text
* Ukrainian-first interface


## Data Source

The project use a reliable source for the official text of the Constitution of Ukraine:

https://zakon.rada.gov.ua/laws/show/en/254%D0%BA/96-%D0%B2%D1%80
